# Manual de Técnico - Micro1475

Manual de procedimientos operativos para técnicos de soporte.

## 1. Gestión de Tickets SAT
- **Listado de Tickets**: Acceso a la cola de trabajo desde el menú lateral.
- **Asignación**: Los tickets pueden ser auto-asignados o asignados por un administrador.
- **Comunicación**: Es obligatorio responder periódicamente al cliente a través del chat interno del ticket para mantenerle informado.

## 2. Tipos de Ticket
- **Consulta**: Dudas técnicas. Se resuelven con información o guiando al usuario.
- **Reparación**: Equipos físicos que llegan al taller. Requieren diagnóstico y presupuesto.

## 3. Conversión a Pedido de Reparación
Esta es la función principal para facturar trabajos de taller.

1. Abre el ticket de tipo "Reparación".
2. Haz clic en el botón verde **"Crear Pedido de Reparación"**.
3. **Mano de Obra**: Introduce el número de horas dedicadas (el sistema calcula automáticamente a 80€/h).
4. **Piezas y Repuestos**:
   - Añade cada pieza utilizada indicando su nombre y el coste para el cliente.
   - Ejemplo: "Pantalla iPhone 13" - 120€.
5. **Generar**: Al confirmar, se creará automáticamente un pedido y una factura asociada. El ticket pasará a estado "En Progreso" y se actualizará el coste total estimado.

## 4. Finalización y Cierre
- Una vez reparado el equipo y verificado el pago del pedido asociado:
- El técnico debe cambiar el estado del ticket a **"Resuelto"**.
- Se debe añadir una nota final detallando la intervención realizada.
- El cliente podrá valorar el servicio una vez el ticket se marque como cerrado.
