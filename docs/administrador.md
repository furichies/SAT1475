# Manual de Administrador - Micro1475

Este manual está dirigido al personal de gestión con permisos de administrador o superadministrador.

## 1. Dashboard Principal
El panel de control ofrece una visión general del negocio:
- Gráficas de ventas y tickets mensuales.
- Estado actual de los pedidos y tickets críticos.
- Métricas de rendimiento de los técnicos.

## 2. Gestión de Productos (Tienda)
- **Añadir Producto**: Permite crear nuevas referencias en el catálogo. Es vital introducir el SKU único y el stock inicial.
- **Categorías**: Gestión de las categorías disponibles en la tienda.
- **Ofertas**: Marcar productos como destacables o en oferta para mostrarlos en la home.

## 3. Gestión de Pedidos
- Visualización de todos los pedidos realizados en la plataforma.
- **Cambio de estado**: Actualizar de "Pendiente" a "Procesando", "Enviado" o "Entregado".
- **Edición**: Permite corregir direcciones de envío o datos del cliente si es necesario.
- **Generación de Albarán/Factura**: Descarga automática en formato PDF.

## 4. Gestión de Clientes
- Acceso a la base de datos de usuarios registrados.
- Capacidad para activar/desactivar usuarios o resetear contraseñas.
- Historial detallado de compras y tickets por cliente.

## 5. Gestión del Personal Técnico
- Alta de nuevos técnicos y asignación de sus especialidades.
- Nivel de técnico (Junior, Senior, Experto) para balanceo de carga.
- Monitorización de tickets asignados a cada empleado.

## 6. Base de Conocimiento (Knowledge Base)
- Creación de artículos de ayuda para clientes.
- Categorización de problemas comunes para reducir el volumen de tickets.
